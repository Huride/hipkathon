시작화면에서 이렇게 설정해주세요!

  • 이름: 원활한 진행을 위해 본명으로 설정해주세요!
  • 캐릭터 설정: edit character 버튼을 눌러 자유롭게 꾸며주세요!
  • 비디오 OFF
  • 마이크 OFF
  • 설정이 모두 끝났다면 Join the gathering 버튼을 눌러 맵에 입장해주세요!

단축키

  • 방향키: 키보드 화살표 혹은 WASD

    타운 내에서 방향을 조정하며 이동할 수 있습니다.

  • Object 사용하기: X

    가까이 다가가면 노랗게 빛나는 물건들은 X키를 눌러 내용을 확인할 수 있습니다.

  • 이모티콘 사용: 숫자 1~6

    숫자를 눌러 이모티콘을 사용할 수 있습니다. 본인의 상태를 표현해보세요!

  • 춤추기: Z
  • 지도 보기: 모니터 옆 지도 아이콘 클릭

    맵 전체 구성과 현재 위치를 볼 수 있습니다.

  • 고스트모드: G

    고스트 모드로 이동하면 물체를 통과할 수 있습니다.

    주로 인원이 꽉 찬 Private Space에 참여하고 싶을 때 사용합니다.

장소별 기능 소개

  • 5칸 이내의 참가자들과는 서로 대화할 수 있습니다.
  • Private Space

Private Space안에 있는 참가자들만 이야기를 함께 나눌 수 있습니다.

고스트 모드 ('G' 버튼) 를 사용하면 이미 꽉찬 프라이빗 공간에 들어가서 다른 참여자들의 이야기를 들을 수 있습니다. 다만, 다른 참가자들은 당신의 목소리를 듣거나 대화할 수 없습니다.

  • Spotlight Space

이 공간에서 발표하면 맵 내의 모든 사람들이 당신의 발표를 들을 수 있습니다. MC와 본인등판 출연자들의 공간입니다.

상대방과 대화&채팅하기

  • Nearby: 가까이 있는 사람들에게 채팅 보내기
  • Everyone: 모든 사람들에게 채팅 보내기
  • 특정 사람에게 채팅 보내기: 보낼 사람 위에서 오른쪽 마우스 클릭 → Message 보내기

화면 공유하기

  • 하단 바 → 모니터 아이콘 클릭 → 공유할 화면 선택 후 공유

상대방 위치 찾기

  • Follow: 자동으로 상대방을 따라 다닐 수 있습니다.
  • Locate on map: 선으로 상대방의 위치를 알려줍니다.
